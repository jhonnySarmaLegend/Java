# Snake and Ladder Game

## Overview
This project implements a console-based Snake and Ladder game. Players compete to reach the final cell (100) by rolling dice and moving across the board while navigating obstacles (snakes) and benefits (ladders).

## How to Run
1. Compile the project using a Java IDE or command line.
2. Execute the `Driver` class.
3. Follow the prompts to enter the number of snakes, ladders, and players.

## Game Rules
- The board has 100 cells.
- Players roll a die to move their pieces.
- Landing on a snake leads down, while landing on a ladder results in climbing up.
- A player wins by landing exactly on cell 100.
